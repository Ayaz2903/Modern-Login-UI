# Modern Login UI

A responsive and modern login page built using **HTML**, **CSS**, and **JavaScript**. This project features a clean authentication interface with password visibility toggle, social login buttons, and a user-friendly design.

##  Features

- Responsive design
- Password show/hide toggle
- Social login buttons
- Clean modern UI
- Mobile-friendly layout

##  Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Purpose of the Project

This project was created to improve frontend development skills and practice building responsive authentication pages without using frameworks. It can also be used as:

- A beginner frontend project
- Portfolio showcase project
- UI/UX practice project
- Base template for authentication system

##  Learning Outcomes

Through this project, I practiced:
- Responsive web design
- DOM manipulation with JavaScript
- CSS styling techniques
- Frontend UI development
- Project structuring for GitHub
